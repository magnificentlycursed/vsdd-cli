# Phase 2a Primer: Test Suite Generation (Red Gate)

## Composition

You are entering Phase 2a (Test Suite Generation / Red Gate). Per the phase-domain composition matrix, load:

- **Quality Engineer** — primary; owns Red Gate authoring + falsifiability check

Plus the always-on baseline. Skill mode.

## Phase-specific discipline

Phase 2a authors **failing tests** against the layer's acceptance criteria from Phase 1c. The Red Gate is the committed test suite that **fails by default** when run against the current (unimplemented or partially-implemented) codebase. The Red Gate is what Phase 2b implementation must turn green.

The Exacting Mentor stance applies: "test that passes against an empty function body" is the failure mode — every Phase 2a test must assert the layer's named behavior, not just liveness. The QE Dim 2 (test falsifiability) discipline: every test answers "what would have to be true of the implementation for this test to fail?" — if the answer is "the function panics" the test is testing liveness, not behavior.

Per-test discipline:
- **Failing-by-default at commit time** — `cargo test` (or equivalent) fails on every test added in this phase
- **Named acceptance criterion** — every test maps to one or more behaviors from DESIGN.md § Behavioral contracts
- **Specific assertion** — the test asserts the layer's named behavior, not just "doesn't crash"
- **Edge case coverage** — empty / null / max-size / unicode / concurrent / partial-failure cases enumerated per behavior
- **Auto-scaffolded skeleton** — the `post-design-md-modification.py` hook auto-scaffolded the test file structure during Phase 1c; this phase fills in test bodies

The draft PR for the layer opens at the first Phase 2a commit (per the Layer-cycle PR discipline). The PR template's manual-tests checklist is auto-generated by `vsdd observe pr-body --layer <N>` from `manual-tests/layer-N.md`.

## Phase-completion criteria

Phase 2a closes when:

- All acceptance criteria from Phase 1c have at least one failing test asserting the behavior
- `cargo test` (or equivalent) returns non-zero (Red Gate is red)
- Draft PR is open for the layer (`check-draft-pr-presence.py` hook validates)
- Manual-test checklist for behaviors that can't automate is committed at `manual-tests/layer-N.md`

Emit `PhaseExited{phase: phase-2a, exit_status: complete, layer: <N>}` at the closing commit. The Red Gate opens Phase 2b.
